# Portfolio

A polished personal portfolio website built with React, Vite, Tailwind CSS, and React Router.

This project demonstrates a modern developer landing page with:
- Responsive navigation and mobile-friendly menu
- Animated hero section with typing effect
- Project showcase cards with interactive GitHub/Demo buttons
- About, Contact, and Projects pages
- Contact form that opens an email draft using `mailto:`
- Smooth visual style with dark theme and gradient accents
